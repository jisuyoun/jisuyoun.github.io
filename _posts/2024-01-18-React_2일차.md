# React 공부 2일차 - JSX

<br />

## JSX

JSX 는 자바스크립트의 확장 문법을 말하는 것으로, 자바스크립트와 XML, HTML 을 합친 것이다.   
###### 예시

```javaScript
1 const element = <h1>Hello, world!</h1>;
```

위 예시는 h1 태그로 둘러싸인 문자열을 element 라는 변수에 저장하는 것이다.

JSX의 역할은 내부적으로 XML, HTML 코드를 자바스크립트로 변환하는 과정을 거치게 되며,    
JSX 코드 사용하지 않을 경우 createElement 를 사용한다. createElement 파라미터로는 type, [props], [...children]이 매개변수로 들어간다.   
###### * children 이란? 엘리먼트가 가지고 있는 자식 엘리먼트를 말한다.   
 **JSX 를 사용하면 가독성이 있으며 Injection Attacks 를 방어하기 쉬워지고, 더 쉽게 작성할 수 있다!**    
###### * Injection Attacks 는 입력창에 문자나 숫자가 아닌 소스코드를 넣어 공격하는 해킹 방식을 말한다.

<br />

## JSX 사용법

```javaScript
const element = <h1>hello, {name}</h1>; 
const element = (
	<div>
		<h1>Hello</h1>
		<h2>World</h2>
	</div>
);
```
